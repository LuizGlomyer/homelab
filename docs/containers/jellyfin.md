# Jellyfin

Jellyfin is a free, open-source entertainment system that runs on the web browser and handles your media collection. It's an alternative to Plex and is fully self-hosted with no mandatory account creation or data collection.

Jellyfin supports:
- Movies, TV Shows, Music (with optional cover art and metadata)
- Multiple user accounts with configurable permissions
- Responsive web interface and native client apps
- Streaming to multiple devices simultaneously
- Automatic media library organization and metadata fetching
- Transcoding for format compatibility
